# C Standard Flags
C has several standards

### C89
```sh
-std=c89
```

### C99
```sh
-std=99
```
Introduced:
```c
for (int i = 0; i < 10; i++)
```

### C11
```sh
-std=c11
```
Very common.

### C17
```sh
-std=c17
```
Mostly bug fixes.

### C23
```sh
-std=c23
```
Latest standard.

Example:
```sh
gcc main.c -std=c23
```
Professional projects commonly use:
```sh
-std=c11
```
or
```sh
-std=c17
```
