# What are compiler Flags?
Flags are options passed to a compiler such as:
```bash
gcc main.c -Wall -Wextra -o app
```
Here:
```bash
-Wall
-Wextra
-o app
```
are all compiler flags:

1. They tell the compiler how to compile the program.
2. Think of them as settings that control;
    - Warnings
    - Errors
    - Optimization
    - Debugging
    - Security
    - Standards
    - Linking
    - Architecture-specific features
