# Data Types in C

Each variable in c must have a data type specified or you will run into errors, C is a statically typed language
so each variable must have the data type.

Data types are as follows:
- Integer (int)
- Float (float)
- Character (char)
- Double (double)

Size of the Data types
- Int (4 bytes)
- float (4 bytes)
- char (1 byte)
- double (8 bytes)
